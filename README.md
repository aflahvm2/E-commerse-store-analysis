# E-commerse-store-analysis
Ecommerce Store Analysis Dashboard

 Project Overview
This project presents an Ecommerce Store Analysis Dashboard built using Microsoft Excel.  
The dashboard provides insights into customer behavior, payment trends, sales performance, and delivery metrics.  

It’s designed to help businesses make data-driven decisions by visualizing key performance indicators (KPIs) and trends.

 Dataset Details
The dataset includes transactional data from an ecommerce platform, containing:
- Order details (order ID, purchase date, payment type)
- Customer information (city, state)
- Payment data (price, total payment)
- Delivery data (shipping days, delivery time)
- Review scores


 Tools & Techniques Used
- Microsoft Excel
  - Pivot Tables & Pivot Charts  
  - Slicers and Filters  
  - Data Cleaning and Preparation  
  - KPI Cards and Conditional Formatting  


 Dashboard Highlights
The dashboard showcases:
- Weekday vs Weekend Payment – 77% of payments occur on weekdays.  
- Top 5 States by Payments – São Paulo leads with the highest payments (0.4M).  
- Average Delivery Days for Pet Shop Orders – Around 11.7 days.  
- Shipping Days vs Review Score – Better reviews correspond to faster deliveries.  
- Year-over-Year Sales Growth – Strong growth from 2016 to 2018.  
- Average Price & Payment Analysis for São Paulo – Avg. price: 97.26, Avg. payment: 154.10.  



Key Insights
- Weekdays dominate the order volume.  
- São Paulo contributes the most to total payments.  
- Higher review scores are linked to shorter shipping durations.  
- Sales show consistent growth year-over-year.
